# How to Add DNS Records in Hostinger

## The Issue: You Need Hostinger Nameservers First!

**To add DNS records in Hostinger, you MUST use Hostinger nameservers, not Vercel nameservers.**

---

## Step 1: Switch to Hostinger Nameservers

1. **In Hostinger DNS / Nameservers page**, you should see:
   - "Use Hostinger nameservers (recommended)" - **Select this!**
   - "Change nameservers" - Don't select this

2. **Select "Use Hostinger nameservers (recommended)"**

3. **Click "Save"** button

4. **Wait a moment** for it to save

---

## Step 2: Find DNS Records Section

After switching to Hostinger nameservers:

1. **Look at the tabs** at the top:
   - **"DNS records"** tab should be active (highlighted in purple)
   - If not, click on **"DNS records"** tab

2. **Scroll down** on the page - there should be a section called:
   - **"Manage DNS records"** or
   - **"DNS Records"** or
   - A table/list of DNS records

3. **You should see fields to add records:**
   - Type dropdown (A, CNAME, TXT, MX, etc.)
   - Name field
   - Points to / Value field
   - TTL field
   - "Add Record" button

---

## Step 3: If You Still Don't See DNS Records

**Try these:**

1. **Refresh the page** (F5 or Cmd+R)

2. **Click on "DNS records" tab** again

3. **Scroll down** - the DNS records section might be below the nameservers section

4. **Look for a button** that says:
   - "Add Record"
   - "Manage DNS"
   - "DNS Records"

5. **Check if there's a dropdown or menu** to switch between nameservers and DNS records

---

## Alternative: Use Different Hostinger Interface

**If you still can't find it, try:**

1. **Go back to main Hostinger dashboard**
2. **Click "Domains"** → **"truckmateslogistic.com"**
3. **Look for:**
   - "DNS Zone Editor"
   - "Advanced DNS"
   - "DNS Management"
   - "DNS Settings"
4. **Click on that** - it should show DNS records interface

---

1. ✅ **Select "Use Hostinger nameservers (recommended)"**
