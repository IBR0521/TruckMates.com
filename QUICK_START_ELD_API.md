# Quick Start: Get ELD API Credentials in 5 Minutes

## 🚀 Fastest Way: KeepTruckin (5 minutes)

### Step 1: Sign Up (2 minutes)
1. Go to **https://keeptruckin.com**
2. Click **"Get Started"**
3. Enter email, company name, phone
4. Verify email

### Step 2: Get API Credentials (2 minutes)
1. Go to **https://developer.keeptruckin.com/**
2. Sign in with your KeepTruckin account
3. Click **"Create New App"**
4. Name it: "TruckMates Integration"
5. Click **"Create"**
6. **Copy** your API Key and API Secret (save them!)

### Step 3: Get Device ID (1 minute)
1. Go to KeepTruckin dashboard
2. Click **"Fleet"** → **"Vehicles"**
3. Click on a vehicle
4. **Copy** the Device ID (number)

### Step 4: Add to Your SaaS
1. Go to your SaaS → **ELD Service** → **Add ELD Device**
2. Enter:
   - Device Name: "Truck #1"
   - Provider: **keeptruckin**
   - Provider Device ID: (from Step 3)
   - API Key: (from Step 2)
   - API Secret: (from Step 2)
3. Click **"Save"**

### Step 5: Test
1. Click **"Sync Now"**
2. Check if data appears ✅

---

## 📋 What You Need

- [ ] KeepTruckin account (free trial available)
- [ ] API Key (from developer portal)
- [ ] API Secret (from developer portal)
- [ ] Device ID (from fleet dashboard)

Once you add the credentials, your ELD service will:
- ✅ Automatically sync data every 15 minutes
- ✅ Pull real-time GPS locations
- ✅ Track HOS logs
- ✅ Monitor violations
- ✅ Use accurate data in IFTA reports

**Total time: ~5 minutes!** 🎉
